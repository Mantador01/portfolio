Mets ici uniquement tes fichiers .obj si tu ne veux pas utiliser de .mtl.

Exemples :
- obj-terrain-1.obj
- obj-terrain-2.obj
- montagne.obj

Ensuite ajoute chaque fichier dans assets/js/terrain-data.js, dans TERRAIN_MODELS.
Le viewer peut appliquer une texture PNG globale déclarée dans TERRAIN_TEXTURES.
